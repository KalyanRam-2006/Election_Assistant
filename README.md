# 🗳️ CivicPulse AI

An advanced, comprehensive platform designed to streamline the electoral process, educate voters, and provide real-time assistance. This project serves as an all-in-one companion for citizens, empowering them with CivicPulse AI, guided voter journeys, and immersive voting simulations.

## ✨ Key Features

- **🤖 Intelligent Local NLP Chatbot**: An offline-capable, context-aware chatbot with fuzzy logic for typo tolerance. It seamlessly handles queries in multiple languages, including English, Hindi, and Hinglish.
- **📍 Polling Booth Locator**: Automated, location-based polling booth discovery integrated directly into the chat interface.
- **🗺️ Guided Voter Journey**: An interactive, step-by-step cinematic guide that educates new and returning voters on the electoral process.
- **🗳️ Mock EVM Simulation**: A highly realistic Electronic Voting Machine simulation to familiarize users with the voting mechanism.
- **📅 Election Timeline UI**: A dynamic visual timeline keeping users informed about crucial election phases and dates.
- **💎 Premium UI/UX**: Built with "Glassmorphism 2.0" aesthetics, featuring vibrant colors, modern typography, and smooth micro-animations powered by Framer Motion.

## 🛠️ Technology Stack

- **Frontend**: React.js, TailwindCSS, Vanilla CSS, Framer Motion
- **Backend**: Node.js, Express.js
- **Intelligence**: Custom Local NLP Engine

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Node.js
- npm (Node Package Manager)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/KalyanRam-2006/Election_Assistant.git
   cd Election_Assistant
   ```

3. **Start Frontend Application:**
   Open a new terminal window:
   ```bash
   cd frontend
   npm install
   npm start
   ```

4. The application will be accessible at `http://localhost:3000`.
